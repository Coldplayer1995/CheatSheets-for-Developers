## Array

    ```python
    # Create array
    np.array([1, 2, 3])
    np.array([[1, 2, 3], [4, 5, 6]])
    np.array([1, 2, 3], ndmin=2)
    np.array([1, 2, 3], dtype=complex)
    ```

Output:

    ```python
    array([1, 2, 3])
    array([[1, 2, 3],
           [4, 5, 6]])
    array([[1, 2, 3]])
    array([1.+0.j, 2.+0.j, 3.+0.j])
    ```
